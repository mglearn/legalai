# Web Deck System

The authoritative specification for this deck system is **`webdeck_instructions_v5.md`** at the project root. Read it in full before building or editing a deck — it is the single source of truth and is kept current as the framework changes.

It covers: the two shared files (`assets/deck-framework.css`, `assets/deck-framework.js`), the blue-and-silver palette and fonts, the 1280×720 slide canvas and skeleton, every slide type and content block, image/photo layout patterns and the WebP optimization step, the navigation chrome (controls, progress bar, exit-to-hub icon), speaker notes and the presenter view, print/PDF export, the headless-Chrome verification workflow, the build checklist, on-slide writing style, resource pages and the language switcher, and the deploy flow.

Do not duplicate those instructions here — update `webdeck_instructions_v5.md` instead, so guidance never diverges.
